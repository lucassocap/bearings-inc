# Bearings Inc - Odoo eCommerce Website

Professional bearing distributor website built with Odoo 17.

## Features
- 20+ bearing products (FAG, SKF, NACHI, NTN, SW, MRC)
- Automated product import with validation
- Netora-inspired design
- Multi-language support (English)
- eCommerce ready

## Quick Start
```bash
docker-compose up -d
python3 scripts/setup.py
```

## Tech Stack
- Odoo 17
- PostgreSQL 15
- Docker
- Python 3.9+
